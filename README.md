
# Global Tourism Analysis Dashboard

This project analyzes global tourism data from 2022 to 2024 using Tableau.

## 📊 Features
- KPI Summary
- Top 10 Most Visited Countries
- Bottom 10 Least Visited Countries
- Global Tourism Map
- Tourism Growth Comparison
- Trend Analysis

## 🛠 Tools Used
- Tableau Public
- CSV Dataset

## 📈 Insights
- Tourism increased significantly from 2022 to 2024
- A few countries dominate global tourism
- Large gap between top and bottom countries

## 📁 Files Included
- Tableau Dashboard (.twbx)
- Dataset (.csv)
- Flowchart & Images
